### 修正内容

**compose.yaml**
- ANTHROPIC_API_KEY環境変数を削除
- claude-code自動起動を停止
- シンプルなbash環境に変更
- `.claude`ディレクトリマウントを削除

```yaml
services:
  claude-code-log:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: claude-code-log
    volumes:
      - .:/workspace
    working_dir: /workspace
    command: ["bash"]
    stdin_open: true
    tty: true
    environment:
      - TERM=xterm-256color
    init: true
```

**Makefile**
- 環境変数チェックを削除
- claude-code関連コマンドを削除
- シンプルな開発環境アクセスに変更

```makefile
# Docker Compose コマンド
up:
	docker compose up -d

# 開発環境アクセス
dev:
	docker compose exec claude-code-log bash
```

### 使用方法
```bash
make up    # コンテナをバックグラウンド起動
make dev   # 開発環境にアクセス
make shell # 同じくシェルアクセス
```

これで純粋な開発環境コンテナとして機能します。
